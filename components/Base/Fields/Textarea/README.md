# Textarea Field

Multi-line text input with Material-styled underline animation and floating label.

## Props
- `config`: FieldConfig — field configuration
- `value`: `string` — text value
- `onChange`: `(value: string) => void`
- `error`: `string | null`

## Example
```typescript
{ name: 'description', type: 'textarea', label: 'Description', rows: 4 }
```

# String Field

Text input with Material-styled underline animation and floating label.

## Props
- `config`: FieldConfig — field configuration
- `value`: `string` — text value
- `onChange`: `(value: string) => void`
- `error`: `string | null`

## Example
```typescript
{ name: 'title', type: 'string', label: 'Title', placeholder: 'Enter title' }
```

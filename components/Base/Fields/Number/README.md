# Number Field

Numeric input with Material-styled underline animation and floating label.

## Props
- `config`: FieldConfig — field configuration
- `value`: `number | null` — numeric value
- `onChange`: `(value: number) => void`
- `error`: `string | null`

## Example
```typescript
{ name: 'price', type: 'number', label: 'Price', placeholder: '0.00' }
```
